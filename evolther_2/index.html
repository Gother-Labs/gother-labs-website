Evölther scrollytelling run walkthrough

Pseudo run contract

01 run():
02 initialize_manifest_and_telemetry()
03 evaluate_seed_and_seed_islands()
04 for generation in budget:
05 worker_plan = schedule_parallel_workers()
06 parent, inspirations = sample_local_archive()
07 proposal = propose_patchproposal(parent, inspirations)
08 vetted = preflight_patchproposal(proposal)
09 candidate = materialize_candidate(vetted)
10 result = evaluate_in_subprocess(candidate)
11 ledger.append_attempt(candidate, result)
12 archive.update_niche(island, candidate, result)
13 if migration_step(): migrate_ring_elite()
14 emit_summary_manifest_and_replay()

01

              
Run contract

              
The run begins before any island, candidate, or evaluator exists.

            


            

              
02

              
Manifest and telemetry

              
The experiment writes its identity, configuration, seed, and evidence surfaces before search starts.

            


            

              
03

              
Seed and islands

              
The seed is measured once, then becomes the starting state of every island.

            


            

              
04

              
Generation loop

              
The seeded system now enters its repeated search budget.

            


            

              
05

              
Parallel workers

              
A generation is a bounded wave of workers, not a single serial attempt.

            


            

              
06

              
Local archive sampling

              
Parent and inspirations are sampled from the island archive, not a global best list.

            


            

              
07

              
Bounded patch proposal

              
The proposer returns a PatchProposal scoped to EVOLVE blocks only.

            


            

              
08

              
Preflight gate

              
Unsafe proposals stop here before any candidate workspace exists.

            


            

              
09

              
Workspace and subprocess

              
The candidate is materialized minimally and judged in a clean subprocess.

            


            

              
10

              
History versus archive

              
Every attempt survives in history, even when it never enters the live archive.

            


            

              
11

              
Niche-local acceptance

              
Only niche winners stay. Thresholds adapt early, then freeze.

            


            

              
12

              
Migration and pressure

              
Islands stay separate except for explicit ring migration and pressure shifts.

            


            

              
13

              
Evidence layer

              
The run closes as evidence: manifest, telemetry, summary, replay.
